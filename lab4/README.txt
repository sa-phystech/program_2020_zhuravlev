Игра "Поймай шарик"

Цель игры: Кликнуть по передвигающейся цели, чтобы заработать очки.

Есть 2 типа цели: Шарик и овал. Шарик двигается плавно по экрану, за клик по нему даётся 1 очко. Овал телепортируется по экрану, его труднее поймать, поэтому за его поимку дают 2 очка.

У игрока есть 3 жизни, за промах отнимается 1  жизнь. Когда у игрока заканчиваются жизни, то игра заканчивается.
